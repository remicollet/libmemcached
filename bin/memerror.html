memerror¶

SYNOPSIS¶

memerror [options] <error code>

Translate a memcached error code into a string.

DESCRIPTION¶

memerror translates an error code from libmemcached into a human readable string.

OPTIONS¶

--help¶

Display help.

--version¶

Display version.

ENVIRONMENT¶

MEMCACHED_SERVERS¶

Specify the list of servers.
